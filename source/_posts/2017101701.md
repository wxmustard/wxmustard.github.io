# ubuntu 网卡问题

## ifconfig 只有lo

* ```bash
  cd /etc/network
  sudo vim interfaces
  # 在interfaces中添加以下语句
  auto eno1
  iface eno1 inet dhcp # dhcp是动态获取IP信息
  # 开启eno1 
  ifconfig eno1 up 
  # 查看IP信息
  ifconfig
  ```

## ifconfig中显示eno1，但是没有inet/地址/广播/掩码

* ```bash
  # 更新IP地址
  sudo dhclient eno1
  # 运行eno1
  sudo ifconfig eno1
  ```
